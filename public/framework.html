Framework

The Telios Framework is composed of diverse applications that empower existing web standards.

Martinelli

Martinelli is an HTTP Server that exposes serial port (RS-232/422) devices as a RESTful web service.

Ramune

Ramune is an Adobe Flash 10 application specializing in Microphone/Camera acquisition and Audio/Video communication and presentation. It is meant to interface entirely through JavaScript and contains no chrome.
