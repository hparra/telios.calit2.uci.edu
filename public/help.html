Help

Installation

Thank you for your interest in Telios. Please keep in mind that you have been invited to try this prototype software that is not yet complete. Due to the nature of the internet, this prototype may not be compatible with certain web browsers, including Microsoft Internet Explorer. Please follow the instructions below in order to optimize your experience with Telios.

Windows Users:

Install Browser

  • Download Firefox 3.5 at http://www.mozilla.com/en-US/products/download.html?product=firefox-3.5.2&os=win&lang=en-US
  • Double-click the setup file to begin installing firefox
  • Follow the on-screen instructions to Install Firefox

Install Flash

  • Download Flash 10 at http://get.adobe.com/flashplayer/
  • Click “Agree and Install Now” to begin downloading and installing Flash
  • Follow the instructions at http://www.adobe.com/products/flashplayer/productinfo/instructions/ if you are having trouble

Start Telios

  • Open Firefox by clicking on the Firefox icon on your desktop, or going to the Start menu and clicking Start, Programs, then choosing Firefox under the Mozilla Firefox
  • Once Firefox is open, go to https://armin.calit2.uci.edu/frescolita/
  • You may be prompted with a screen like this:
  • Click “I Understand the Risks”, then click “Add Exception”
  • A window like this should pop up:
  • Click “Get certificate” and then click “Confirm Security Exception”
  • Once this is done and the page has loaded, press the start button. Note: If you receive messages about not having flash or the correct browser, please review the on-screen instructions or refer back to the first two parts above.

Browser Compatibility

  • Firefox 3.0 and higher
  • Safari 4
  • Chrome
